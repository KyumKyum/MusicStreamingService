5. User 클래스 생성
> 보통 디비에서 가져온 정보를 통해 User 클래스를 만들고, 
> 업데이트 등의 사항이 있을 때 그 유저클래스의 값을 변경시키고,

> 불변객체를 유지하고 싶다면 업데이트시 변경 내용이 포함되는 새로운 User 인스턴스를 생성한다.
